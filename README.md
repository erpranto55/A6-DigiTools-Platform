# DigiTools – Digital Tools Buying Website

## Project Overview

DigiTools is a modern digital tools marketplace where users can browse, select, and purchase premium digital products

## Live Site

[DigiTools](https://a6-digi-tools-platform-tan.vercel.app/)

## GitHub Repository

[GitHub-DigiTools](https://github.com/erpranto55/A6-DigiTools-Platform)

## Technologies Used

React.js<br>
Tailwind CSS<br>
DaisyUI<br>
JavaScript (ES6+)<br>
JSON (for product data)<br>
React Toastify<br>

## Key Features

### 1. Add to Cart System

--> Users can add products to cart<br>
--> Cart count updates instantly in navbar<br>
--> Prevents duplicate product addition<br>

### 2. Remove & Checkout Functionality

--> Remove individual items from cart<br>
--> Shows total price dynamically<br>
--> "Proceed to Checkout" clears all items<br>

### 3. Toast Notifications

--> Shows alerts for:<br>
--> Add to cart<br>
--> Remove item<br>
--> Checkout<br>
--> Built using React Toastify<br>

## UI Features

Fully responsive design (Mobile, Tablet, Desktop)<br>
Modern gradient-based UI<br>
Interactive buttons with hover effects<br>
Product cards with tags (Popular, New, Best Seller)<br>

## Product Data

Dynamic product loading using JSON<br>
Each product includes:<br>
--> Name<br>
--> Description<br>
--> Price<br>
--> Features<br>
--> Tag<br>
--> Icon<br>

## Optional Features Implemented

Button text changes to "Added" after clicking<br>
Button color changes dynamically<br>
Cart count sync with UI<br>
Prevent duplicate product selection<br>
