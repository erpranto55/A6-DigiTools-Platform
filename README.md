# DigiTools – Digital Tools Buying Website

## Project Overview

DigiTools is a modern digital tools marketplace where users can browse, select, and purchase premium digital products

## Live Site

[DigiTools](https://a6-digi-tools-platform-tan.vercel.app/)

## GitHub Repository

[GitHub-DigiTools](https://github.com/erpranto55/A6-DigiTools-Platform)

## Technologies Used

React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
JSON (for product data)
React Toastify

## Key Features

### 1. Add to Cart System

--> Users can add products to cart
--> Cart count updates instantly in navbar
--> Prevents duplicate product addition

### 2. Remove & Checkout Functionality

--> Remove individual items from cart
--> Shows total price dynamically
--> "Proceed to Checkout" clears all items

### 3. Toast Notifications

--> Shows alerts for:
--> Add to cart
--> Remove item
--> Checkout
--> Built using React Toastify

## UI Features

Fully responsive design (Mobile, Tablet, Desktop)
Modern gradient-based UI
Interactive buttons with hover effects
Product cards with tags (Popular, New, Best Seller)

## Product Data

Dynamic product loading using JSON
Each product includes:
--> Name
--> Description
--> Price
--> Features
--> Tag
--> Icon

## Optional Features Implemented

Button text changes to "Added" after clicking
Button color changes dynamically
Cart count sync with UI
Prevent duplicate product selection
